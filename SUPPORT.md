**Wiki:** [akku-craft.usbverkehrtherum.de/wiki](https://akku-craft.usbverkehrtherum.de/wiki)

- Contribution guidelines

## Reporting Issues

If you've found a bug or want to request a feature, please open an issue in the relevant repository using the appropriate issue template. Make sure to:

1. Search existing issues first to avoid duplicates.
2. Include a clear description of the problem or suggestion.
3. Attach relevant logs, screenshots, or files where applicable.
